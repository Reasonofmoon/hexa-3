# M1-diagnosis 실습 가이드: 소매업 AI 진단 프롬프트

## 오늘 실습 목표
1. 소매/유통 업계의 AI 도입 현황을 진단한다
2. 10개의 전문 프롬프트로 업계별 맞춤 분석을 수행한다
3. AI 전환 로드맵을 수립하고 실행 계획을 수립한다

---

## 🟢 Starter: 프롬프트만으로 (코딩 없음)

아래를 ChatGPT/뤼튼에 그대로 붙여넣고, `[]` 부분만 바꾸세요:

```
당신은 소매/유통 업계 AI 컨설턴트입니다.
우리 회사(업종: [패션의류], 규모: [매장 10개])의 AI 도입 현황을 진단해주세요.
- 현재 업무: [재고관리, 고객응대, 판매관리]
- 문제점: [재고부족, 고객이탈, 판매향상]
이 기준으로 AI 도입 우선순위 3가지를 제안해주세요.
```

## 🔵 Professional: AI 진단 프롬프트 10개

### 프롬프트 1: 재고 관리 최적화
```
당신은 소매업 재고 관리 AI 전문가입니다.
[대형마트]의 재고 데이터를 분석하여 AI 기반 재고 최적화 방안을 제시해주세요.
- 현재 재고 부족률: [15%]
- 과잉 재고 비율: [8%]
- 목표: 재고 부족률 5% 이하, 재고 회전률 20% 향상
수요 예측 모델과 자동 발주 시스템 구축 계획을 포함해주세요.
```

### 프롬프트 2: 고객 행동 분석
```
당신은 소매업 고객 분석 AI 전문가입니다.
[온라인 쇼핑몰]의 고객 행동 데이터를 분석하여 개인화된 마케팅 전략을 수립해주세요.
- 평균 구매 전환율: [2.5%]
- 고객 이탈률: [25%]
- 목표: 전환율 5% 이상, 이탈률 15% 이하
고객 세분화 모델과 추천 시스템 구축 방안을 제시해주세요.
```

### 프롬프트 3: 가격 전략 최적화
```
당신은 소매업 가격 전략 AI 전문가입니다.
[전자제품 리테일러]의 가격 정책을 분석하여 동적 가격 책정 전략을 제안해주세요.
- 현재 가격 경쟁력: [경쟁사 대비 3% 높음]
- 할인 프로모션 효과: [판매량 15% 증가]
- 목표: 수익률 유지하며 판매량 20% 증가
경쟁사 분석과 시장 수요 기반 가격 최적화 모델을 제시해주세요.
```

### 프롬프트 4: 매장 운영 효율화
```
당신은 소매업 매장 운영 AI 전문가입니다.
[체인카페]의 매장 운영 데이터를 분석하여 효율화 방안을 제시해주세요.
- 평균 고객 대기 시간: [8분]
- 직원 생산성: [시간당 20건 처리]
- 목표: 대기 시간 3분 이하, 생산성 30% 향상
매장 배치 최적화와 직원 스케줄링 AI 시스템을 구축해주세요.
```

### 프롬프트 5: 공급망 관리
```
당신은 소매업 공급망 AI 전문가입니다.
[식료품 슈퍼마켓]의 공급망을 분석하여 최적화 방안을 제시해주세요.
- 현재 납품 지연율: [12%]
- 물류 비용: [매출의 8%]
- 목표: 납품 지연율 5% 이하, 물류 비용 6% 이하
공급망 예측과 자동 주문 시스템 구축 계획을 제시해주세요.
```

### 프롬프트 6: 마케팅 ROI 최적화
```
당신은 소매업 마케팅 AI 전문가입니다.
[패션 브랜드]의 마케팅 데이터를 분석하여 ROI 최적화 전략을 수립해주세요.
- 현재 마케팅 ROI: [1:3.2]
- 고객 획득 비용: [25,000원]
- 목표: ROI 1:5 이상, 획득 비용 15,000원 이하
채널별 효과 분석과 자동 예산 배분 시스템을 구축해주세요.
```

### 프롬프트 7: 고객 서비스 자동화
```
당신은 소매업 고객 서비스 AI 전문가입니다.
[가전제품 리테일러]의 고객 서비스를 분석하여 자동화 방안을 제시해주세요.
- 평균 응답 시간: [4시간]
- 고객 만족도: [78점/100점]
- 목표: 응답 시간 30분 이내, 만족도 90점 이상
챗봇 시스템과 자동 응대 프로세스를 구축해주세요.
```

### 프롬프트 8: 매출 예측 및 분석
```
당신은 소매업 매출 예측 AI 전문가입니다.
[뷰티 스토어]의 매출 데이터를 분석하여 정확한 예측 모델을 구축해주세요.
- 현재 예측 정확도: [75%]
- 계절별 변동성: [±30%]
- 목표: 예측 정확도 90% 이상
시계열 분석과 외부 요인 반영 모델을 제시해주세요.
```

### 프롬프트 9: 리스크 관리 및 사기 탐지
```
당신은 소매업 리스크 관리 AI 전문가입니다.
[온라인 마켓플레이스]의 거래 데이터를 분석하여 사기 탐지 시스템을 구축해주세요.
- 현재 사기 거래율: [0.8%]
- 정상 거래 오탐률: [5%]
- 목표: 사기 거래율 0.2% 이하, 오탐률 2% 이하
이상 거래 패턴 탐지와 실시간 경고 시스템을 설계해주세요.
```

### 프롬프트 10: 옴니채널 통합 전략
```
당신은 소매업 옴니채널 AI 전문가입니다.
[복합 채널 리테일러]의 옴니채널 전략을 수립해주세요.
- 온/오프라인 연계율: [35%]
- 고객 경험 일관성: [65점/100점]
- 목표: 연계율 60% 이상, 일관성 85점 이상
채널 통합 데이터 분석과 개인화된 고객 경험 설계를 제시해주세요.
```

## 🔥 확장 과제

**과제 1 — 경쟁사 벤치마킹**
```
[동종 업계 3개사]의 AI 도입 사례를 분석하여 성공 요인과 교훈을 도출해주세요.
도입 기술, 투자 규모, 성과 지표를 비교 분석하고 우리 회사에 적용 가능한 모델을 제안해주세요.
```

**과제 2 — ROI 분석 모델**
```
AI 도입을 위한 투자 대비 수익 분석 모델을 구축해주세요.
도입 비용, 운영 비용, 기대 효과를 3개년 단위로 예측하고
손익분기점과 순현재가치(NPV)를 계산해주세요.
```
